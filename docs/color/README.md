# Color Docs

Color documentation is grouped by workflow area:

- `architecture/` contains pipeline ADRs, audits, and color-management plans.
- `fixtures/` contains fixture and measurement harness documentation.
- `runtime/` contains parity, ABI, and UI runtime validation notes.
- `selective-color/` contains selective color and skin tone workflow docs.
- `workflow/` contains user-facing color workflow decisions.
