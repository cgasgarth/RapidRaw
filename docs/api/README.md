# API Docs

This folder is grouped by API ownership:

- `guide/` contains broad developer-facing API guidance.
- `commands/` contains edit, render, and mask command contracts.
- `schema/` contains schema package and Tauri validation boundaries.
- `surface-coverage/` contains UI/API coverage reports and their source data.

Keep new API docs near the surface they describe. Root-level files should stay
rare and only cover cross-cutting API architecture.
