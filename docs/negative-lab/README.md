# Negative Lab Docs

Negative Lab documentation is grouped by ownership:

- `architecture/` contains ADRs, conversion architecture, and processing model
  notes.
- `fixtures/` contains fixture provenance, scanner/camera input profile, and
  measured profile documentation.
- `presets/` contains built-in preset policy and metadata contracts.
- `ui-workflows/` contains workspace, roll, frame, sampling, and QC UI specs.
- `validation/` contains Negative Lab validation contracts and schemas.
