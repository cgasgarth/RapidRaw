# Tooling Docs

Tooling notes are grouped by active maintenance area:

- `bun/`, `dependencies/`, `frontend/`, `git-hooks/`, `local-checks/`, `rust/`,
  and `typescript/` contain current tooling contracts and audits.
- `linting/` contains current lint and static-analysis policy.
- `eslint-archive/` keeps older ESLint migration notes that still explain
  historical decisions after the linting cleanup.

Prefer updating the current-area docs over adding new root-level tooling notes.
