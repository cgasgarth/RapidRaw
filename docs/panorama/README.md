# Panorama Docs

Panorama documentation is grouped by implementation concern:

- `architecture/` contains stitcher audits, consult output, and backend
  capability planning.
- `artifact-contracts/` contains sidecar and artifact contracts.
- `opencv/` contains OpenCV evaluation, packaging, and CI promotion notes.
- `projection/` contains projection, boundary, exposure, and multi-row design.
- `validation/` contains validation contracts and private decode smoke notes.
